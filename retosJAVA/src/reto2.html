Explicación de los Eventos:

  • click: Se activa cuando se hace clic en un elemento.
  • mouseover: Se activa cuando el mouse pasa por encima de un elemento.
  • keydown: Se activa cuando se presiona una tecla en un campo de texto.
  • keyup: Se activa cuando se suelta una tecla en un campo de texto.
  • submit: Se activa cuando se envía un formulario.
  • focus: Se activa cuando un elemento recibe el foco.
  • blur: Se activa cuando un elemento pierde el foco.
  • change: Se activa cuando cambia el valor de un elemento.
  • load: Se activa cuando la página se ha cargado completamente.
  • resize: Se activa cuando se cambia el tamaño de la ventana del navegador.
